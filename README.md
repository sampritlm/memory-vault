# Memory Vault - Digital Time Capsule

A full-stack web application that allows users to send messages to their future self or others that unlock at a specific date and time.

## Features

- 🔐 User authentication (Login/Signup)
- 📝 Compose and seal messages with future unlock dates
- 🔒 Messages remain locked until the specified date
- 📊 Dashboard showing sent and received messages
- 📱 Fully responsive design
- 💾 Local storage for demo (no backend required)

## Tech Stack

- **Frontend:** React.js, Vite, Tailwind CSS
- **Routing:** React Router v6
- **Icons:** Lucide React
- **Date Handling:** date-fns, react-datepicker
- **Notifications:** React Hot Toast
